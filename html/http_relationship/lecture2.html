それぞれのシステムのやり取りの内容に関して

ここでは演習で得られた学習を深いものにするため、ウェブサーバ、ブラウザ　や　データベースの間でやり取りされる情報について解説を行います

HTTPとは

WebブラウザとWebサーバの間でHTMLなどのコンテンツの送受信に用いられる通信プロトコルである。

ブラウザからサーバへデータの取得を要求する際に送信するデータは、HTTPリクエストと呼ばれている。

逆に、サーバからブラウザへ応答を送信する、または要求されたデータを送信する際はHTTPレスポンスと呼ばれる

SQLとは

SQLとはデータベースにおいて、データの操作や定義を行うための言語である。

　データベースに保存した情報に関して問い合わせる時、サーバはデータベースにSQLを送信します

SELECT * FROM user WHERE id = 'yamada';

このメッセージでは　idの項目の中に　yamada が入っているデータを全て選択してデータベースから呼び出すことを表しています。
